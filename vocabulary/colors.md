# Lista básica

- grå -> gris
- svart -> negro
- vit -> blanco
- lila -> morado, purpura
- rosa -> rosa
- röd -> rojo
- grön -> verde
- blå -> azul
- orange -> naranja
- gul -> amarillo
- gyllene -> dorados
- brun -> marrón
- färgglad -> colorido, de colores vivos

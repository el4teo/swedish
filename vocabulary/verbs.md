# My own classification

## Basics

- har -> tener, poseer
- tycker om -> gustar

- gör -> hacer
- arbetar -> trabjar


- vill (ha) -> querer
- behöver -> necesitar
- kan -> poder (capacidad de hacer algo)
- måster -> deber

## Debatir

- pratar -> hablar
- vet -> saber
- förstår -> entender

- frågar -> preguntar
- svarar -> responder

- påstår -> afirmar
- nekar -> negar

## Moverse

- stannar -> quedarse, detenerse
- faller -> caer, descender
- går -> ir

## Cambio

- blivit -> convertirse

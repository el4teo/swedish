- jag skulle -> me gustaría

- för -> para

# Argumentar

- men -> pero
- även om -> aunque
- att -> que

- endast -> solo
  - den enda personen -> la única persona

# Relaciones

- vän -> amigo


# Pertenencia

- egna -> propio

# Cantidad

- mycket -> mucho

# Frecuencia

- aldrig -> nunca

- dagligen -> diariamente
- veckovis -> semanalmente

- dag -> día
- natt -> noche

- minut -> minuto

- timma -> hora

- vecka -> semana

- månad -> mes
  - månader

# Durante la comida

## Utencilios

- en kniv -> un cuchillo
  - kniv`ar` -> cuchillos
- en sked -> una cuchara
- en gaffel -> un tenedor
  - gaff`lar` -> tenedores

- en tallrik -> un plato
  - tallrik`ar` -> platos
  - tallrik`arna` -> los platos

# En la oficina

- ett skribord -> un escritorio

- en dator -> a computer

- en stol -> una silla
  - stol`ar` -> sillas
  - stol`arna` -> las sillas

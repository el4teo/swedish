# Releated to furniture

## Places

- ett hus -> una casa
  - hus`et` -> la casa
  - hus`en` -> las casas
- en lägenhet -> un apartamento


- rummen -> los cuartos
- sovrummen -> los dormitorios
- ett kök -> una cocina
  - kök`et` -> la cocina
  - kök`en` -> cocinas
